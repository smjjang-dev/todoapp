Todo List

Supabase 기반 멀티유저 Todo List 앱. 회원가입 후 로그인하면 나만의 할 일 목록을 관리할 수 있고, 다른 사용자의 할 일은 보이지 않습니다.

배포 사이트 바로가기 Android 설치 파일 (todoApp.apk)

APK는 배포 사이트를 그대로 띄우는 TWA 방식이며, Play 스토어 서명이 아니라 사이드로드(출처를 알 수 없는 앱 설치 허용)로만 설치할 수 있습니다.

주요 기능

🔐
이메일 회원가입·로그인
이메일/비밀번호로 가입하고 닉네임이 화면에 표시됩니다.
🌐
소셜 로그인
Google/GitHub 계정으로 가입과 로그인을 동시에 처리합니다.
🚦
중요도 지정
할 일 추가 시 상/중/하 지정, 등록 후에도 언제든 변경 가능합니다.
✅
완료 시각 기록
완료 체크 시 완료 시각이 자동으로 기록됩니다.
⠿
드래그로 순서 변경
드래그 핸들을 이용해 할 일 순서를 자유롭게 바꿀 수 있습니다.
🔒
사용자별 데이터 격리
Supabase RLS로 내 할 일은 나만 볼 수 있습니다.
🌓
라이트/다크 모드 토글
"Todo List" 제목 옆 해/달 아이콘으로 전환하며, 선택한 테마는 기기에 저장되어 다음 방문에도 유지됩니다.
🔓
아이콘 버튼
로그인·로그아웃·할 일 추가 버튼이 자물쇠/＋ 아이콘으로 표시됩니다.

화면으로 보는 사용법

1회원가입

닉네임, 이메일, 비밀번호(6자 이상)를 입력하고 가입하거나, Google/GitHub 버튼으로 바로 가입할 수 있습니다.

2로그인

가입한 이메일/비밀번호로 로그인하거나, Google/GitHub 계정으로 바로 로그인합니다.

3할 일 목록 (빈 상태)

로그인하면 닉네임이 표시되고, 아직 할 일이 없으면 안내 문구가 보입니다.

4할 일 등록 및 관리

할 일을 추가할 때 중요도를 선택할 수 있고, 각 항목의 중요도 배지를 눌러 나중에 바꿀 수도 있습니다. 완료 체크하면 완료 시각이 함께 표시됩니다.

사용 순서 요약

1. 회원가입 → 2. 로그인 → 3. 할 일 입력 후 중요도 선택해 추가 → 4. 완료되면 체크박스 클릭 → 5. 필요하면 중요도 배지를 눌러 재분류 → 6. 드래그 핸들로 순서 변경 → 7. 로그아웃
